Neologia is packaged as a #lang in the Racket programming language, and can be installed as part of the Racket ecosystem.

Instructions

  1. First, make sure you have a relatively recent version of Racket installed.
  2. Open a terminal and run raco pkg install neologia.
  3. Run your files using racket <mylexicon.rkt> command, or open them in Dr. Racket and hit the play button.
