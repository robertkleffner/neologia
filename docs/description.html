Usage

Neologia is a language extension within the Racket ecosystem. After installing Racket, you can run Neologia files as if they were regular Racket source files, both from the command line and from the Dr. Racket IDE. To run a Neologia file fom the command line, just type racket <path/to/my/file>. To run a file from Dr. Racket, just open it up and hit the play button. In both options, Neologia will display the full path to the successfully generated output dictionary.
